Prêt à démarrer ?

Installez NudgeBot en local sur votre PC et lancez le serveur de développement.

Clonez le projet, installez les dépendances et lancez le projet :

git clone https://github.com/QuenumGerald/NudgeBot.git
cd NudgeBot
npm install

# Configurer les variables d'env
cp backend/.env.example backend/.env

# Puis lancer
npm run dev
