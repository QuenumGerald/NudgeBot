Fonctionnalités clés

Conçu pour être rapide, extensible et respectueux de vos données.

Installation en 1 clic

Que ce soit sur votre PC personnel pour un usage local ou sur votre serveur Docker pour une équipe, NudgeBot se déploie en quelques secondes avec une seule commande.

Contrôle total & Privé

Vos clés d'API et vos conversations sont stockées localement sur votre machine. Aucun serveur intermédiaire ne lit vos données sensibles.

Extensibilité MCP Infinie

Connectez NudgeBot à vos outils favoris : Google Calendar, bases de données, systèmes de fichiers ou outils personnalisés grâce au protocole standard MCP.

Mémoire par compression IA

Fini les pertes de contexte lors des longues conversations. Notre système de compression intelligente résume l'historique pour ne jamais perdre le fil.

Prêt à démarrer ?

Installez NudgeBot en local sur votre PC et lancez le serveur de développement.

Téléchargez l'application installable pour votre système d'exploitation :

Nouveau : Assistant de configuration intégré au premier démarrage pour configurer vos clés API en toute simplicité.

Linux (AppImage / .deb) Architecture x64 macOS (.dmg) Apple Silicon & Intel Windows (.exe) Installateur standard

Installez les dépendances et lancez le serveur local :

git clone https://github.com/QuenumGerald/NudgeBot.git
cd NudgeBot
./install.sh
npm run dev
