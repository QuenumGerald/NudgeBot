Prêt à démarrer ?

Installez NudgeBot en local sur votre PC ou déployez-le sur votre propre serveur Docker en une seule commande.

Clonez le projet et lancez le script d'initialisation :

git clone https://github.com/QuenumGerald/NudgeBot.git
cd NudgeBot
npm run init-setup
